These are mini projects of the course on c++ data structures in:
https://www.youtube.com/watch?v=h-GLNodQ1Bc&list=PL6oj8ZA5xT3CSeMEvLLB7iHVkty5c34Sh

1)Recursion(Recursion.cpp):

- Implementing a count down function with two versions: Recursive vs Non-recursive

2)Factorial Implementation(Factorial.cpp):

- Implementing factorial function using:
	->for loop
	->while loop
	->Recursion loop

3)Linear Search(Linear_search_O_n.cpp):
	
- Implementing linear search with complexity of O(n) and computes the search time

4)Binary Search(Linear_search_O_Log_n.cpp):
- Implementing binary search with complexity of O(Log(n)) and computes the search time
- Array should be sorted before using binary search
